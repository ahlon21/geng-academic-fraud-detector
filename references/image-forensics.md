# 图像取证手册（第一式 / 第三式深化）

> 这是耿同学最经典的战场。Claude 以**视觉方式**理解 PDF 里的图，
> 能做"肉眼级"的复用/拼接判断，但做不了像素级 ELA。下面给出系统化的查图方法
> 与诚实的能力边界。

先用 `scripts/extract_pdf.py` 把图取出来再看：
```bash
python3 scripts/extract_pdf.py images paper.pdf -o figs/    # 导出内嵌位图
python3 scripts/extract_pdf.py render paper.pdf -o pages/ --dpi 200   # 整页高清渲染
```
然后用 `Read` 工具逐张打开 `figs/` 与 `pages/` 里的 PNG 仔细比对。

## Bik 三类图像重复（Elisabeth Bik 分类法）

| 类型 | 名称 | 特征 | 查法 |
|------|------|------|------|
| Type I | 简单复用 | 同一张图原封不动用于两处不同实验 | 比对面板是否完全相同 |
| Type II | 移位复用 | 同一图经平移/旋转/翻转/裁剪后复用 | 看可旋转对齐的特征点、共有的污点/噪声 |
| Type III | 篡改复用 | 复制图内某区域覆盖另一区域（克隆图章） | 找重复的背景斑块、不自然的矩形边界 |

**通用查图清单**：
- 逐一遍历所有 figure/subfigure，把"声称不同实验但长得像"的面板配对。
- 重点：Western blot、凝胶电泳、显微镜（HE/IHC/IF）、流式细胞、菌落平板。
- 找**共有的不规则特征**：同一个划痕、气泡、污点、灰尘——它们是图像的"指纹"，
  独立实验不可能共享。
- 检查 loading control（β-actin、GAPDH、Tubulin）是否在不同条件下完全相同。
- 对照图注声称的条件与图像是否匹配（面板与"全视网膜/全切片"图是否对得上）。

## Western blot / 凝胶专项（第三式：拼接）

- **泳道间垂直分界线**：曝光/背景在某条竖线突变 = 拼接痕迹。
- **背景灰度突变**：同一条 blot 不同区域噪声纹理明显不同。
- **条带形状完全一致**：不同样本的条带轮廓像素级雷同 → 复制粘贴。
- **矩形"补丁"**：某条带周围有亮度不同的方块（PS 擦除/植入）。
- **缺失分子量 marker** 或 marker 与条带不匹配。
- **过度均匀的背景**：真 blot 有自然的不均匀；"太干净"反而可疑。

提升肉眼判别力的技巧（提示用户用图像软件）：
- 调高对比度 / 反相 / 拉伸直方图，隐藏的拼接缝与克隆块会"跳出来"。
- 用 ImageJ 的 "Find Edges" 或差值叠加比对两张疑似复用的图。

## 其他图像类型

- **显微/IF 图**：背景细胞排布、自发荧光斑点是否雷同；放大倍数标尺是否前后矛盾。
- **流式细胞 dot plot**：点云分布是否在"不同样本"间完全一致。
- **统计图（柱状/折线）**：误差棒是否过于规整；散点是否"教科书式"无噪声；
  从图里读数反推是否与正文表格数值冲突。
- **图表数据反提取**：必要时从柱状图/散点图目测读数，回到第二式做统计自洽检验。

## 能力边界（必须如实告知用户）

Claude 的图像分析是**视觉理解级**，**不是**像素取证：
- ❌ 无法做 ELA（Error Level Analysis）、JPEG 压缩痕迹分析、EXIF 元数据提取。
- ❌ 无法做亚像素级自动比对、傅里叶域周期性检测。
- ✅ 能识别明显的复用/旋转/拼接、面板与图注矛盾、共有特征点。

对需要像素级确证的发现，报告中标注：
> "建议用 ImageTwin / Proofig / Forensically / ImageJ 做像素级复核。"

并把严重程度定为"疑似（🟠）"而非"实锤（🔴）"，除非有图注矛盾等独立佐证。

## 与外部工具的衔接
- **ImageTwin.ai / Proofig**：商业图像查重，期刊在用，可比对数据库内重复。
- **Forensically (29a.ch)**：免费在线 ELA / 克隆检测 / 噪声分析。
- **ImageJ/Fiji**：开源，做对比增强、差值、边缘检测。

把这些写进报告的"建议后续行动"，体现严谨与边界意识。
